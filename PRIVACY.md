# Privacy Policy for Upstage Document AI Plugin

**Last Updated:** December 4, 2024

## Data Collection and Usage

This plugin interfaces with the Upstage API to provide document parsing and information extraction services.

### What Data We Collect

- **Document Files:** Files uploaded by users for processing (PDFs, images, office documents)
- **API Credentials:** Upstage API keys provided by users for authentication
- **Processing Parameters:** User-defined schemas and output format preferences

### How We Use Your Data

1. **Document Processing:** Files are sent to Upstage API for parsing and information extraction
2. **Temporary Caching:** Processed results are cached temporarily in memory (max 1 hour) to improve performance
3. **No Permanent Storage:** We do not permanently store any user files or processed content

### Data Sharing

- **Upstage API:** Document files and processing requests are transmitted to Upstage's servers for processing
- **No Third-Party Sharing:** We do not share your data with any other third parties

### Data Security

- API keys are transmitted securely via HTTPS
- Cached data is stored in-memory only and automatically expires
- All network communications use encrypted connections

### User Rights

Users have the right to:
- Access their API usage through Upstage Console
- Delete their API keys at any time
- Request information about their data processing

### Compliance

For more information about Upstage's data handling, please refer to [Upstage Privacy Policy](https://www.upstage.ai/privacy-policy).
